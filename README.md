# Flare

Flare is a recovery-support app concept focused on helping users interrupt compulsive or high-risk moments with predefined disruptive actions, reflection prompts, and eventually trusted support-group escalation.

## Current stage

Project bootstrap / vision direction.

## Initial direction

- V0: solo-use support flow without Telegram support.
- V1: trusted group escalation, likely via Telegram or another messaging integration.
- Future: community/group support for people who do not already have a trusted support network.
